# multerimageupload

This repo is about creating a image-upload-app using node.js.

## Tech Stack

**Client:** Ejs

**Server:** Node, Express, Multer

**Database:** Mongodb

    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
